# RecebeFácil

Sistema de gestão de contas a receber com cobrança via WhatsApp.

## Sobre o projeto

O RecebeFácil permite cadastrar clientes, registrar valores devidos e controlar contas a receber. Conta com lembretes de cobrança e envio de mensagens via WhatsApp com um clique.

## Funcionalidades

- Cadastro de clientes
- Registro de contas a receber
- Controle de recebimentos
- Lembretes de cobrança
- Cobrança via WhatsApp

## Tecnologias

- Frontend: HTML, CSS
- Backend: Node.js
- Banco de dados: a definir
